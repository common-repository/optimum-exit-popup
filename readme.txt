== Description ==
Optimum Exit Popup is a simple and light weight exit popup plugin. Using this plugin, you can show popup with information or a subscription form to your website visitors. This is the most easier and user friendly Exit Popup Plugin released in WordPress plugin directory, so far.

== Installation ==
* Download and install this plugin via wordpress directory,
* Once installed, navigate to Optimum Popup to enable the popup.
